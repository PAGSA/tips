Contribute to TIPs!

The goal of TIPs is to provide a quick reference for new (and old!) astronomers to get a feel for what telescopes exist, the differences between them, and what they tend to be used for. TIPs would not be possible without the knowledge of experts from around the world, so if you are especially familiar with an instrument you have used or designed, please consider making an addition to the list! The website is populated from a single JSON file on our Github which contains the information for each telescope. If you wish, you may add information directly using the steps outlined there. Otherwise, use the forms below to submit information which we will format into the JSON file for you.
